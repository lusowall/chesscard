# Contribution

Merci de vouloir contribuer !

## C'est quoi ce projet ?

C'est un projet HTML qui transforme ton profil **chess.com** en une **carte de joueur comme sur FIFA** — rating, stats, aperçu des derniers matchs, le tout stylisé comme une vraie carte de footeux.

## Comment contribuer ?

### Les branches, c'est important

- **`dev`** = là où tu fais tes trucs, testes tes idées, commit ton code en cours
- **`main`** = le travail finalisé, testé et prêt. C'est la vraie prod

### Le workflow

1. Fork le repo (ou crée une branche depuis `dev`)
2. Bosse sur **`dev`** (ou une branche feature depuis `dev`)
3. Teste ton truc localement
4. Quand c'est bon → pull request vers **`dev`**
5. Une fois que `dev` est stable → merge vers `main` (version finale)

### Pas de direct sur main

On push **jamais** directement sur `main`. C'est sacré. Toujours passer par `dev`.

## Types de contributions possibles

- Fix de bugs
- Nouvelles features (design de cartes, intégration de stats, etc.)
- Amélioration de la doc
- Upgrades visuels/UI

## Avant de committer

- Vérifie que ça fonctionne sur ton local
- Teste les cas edge (profil sans matchs, stats vides, etc.)
- Fais des commits clairs et lisibles

## Questions ?

Ouvre une issue ou contacte l'équipe. On répond vite.
